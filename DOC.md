Necesito crear un PWA para llevar el control de inventario de algunos pedidos de comidas familiares que se hacen en la casa.
- Necesito llevar el control de clientes, nombre, telefono y agrega un URL google maps (esto para hacer entrega del producto)
- Necesito llevar el control de productos, nombre, precio y cantidad
- Necesito llevar el control de pedidos, cliente, producto, cantidad, precio, total
- Necesito llevar el control de pagos (parcial, completo, pendiente, etc), cliente, producto, cantidad, precio, total
- Necesito llevar el control de inventario, producto, cantidad, precio, total
- Necesito llevar el control de gastos, concepto, monto, fecha
- Necesito llevar el control de ingresos, concepto, monto, fecha

El producto se crea a base de materia prima y no todo el tiempo se crea la misma cantidad, entonces necesito crear una especie de "Lotes" donde pueda agregar lo que se gasto en la compra de esos ingredientes para ese lote.
Necesito levantar una lista de pedidos (la lista debe tener una fecha) porque en base a esa lista es que se crean los lotes y por ende la cantidad de productos que se van a vender. En esta lista se enlaza o se crea el cliente y el producto.

Necesito que el sistema sea responsive y funcione bien en dispositivos móviles.
La base de datos a utilizar sera firebase.
Necesito que el sistema pueda crear una pequeña empresa, y que luego de crearla a los administradores se les envie una invitacion o link para que puedan administrar la empresa, pedidos, materia prima, etc. Solo aquellos con invitacion se puede crear el usuario (login de google)

El sistema se alojara en netlify entonces que se hagan los push automaticos cuando se hagan cambios en el codigo.
